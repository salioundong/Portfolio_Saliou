Concept

This website is a personal portfolio that showcases my projects and skills. It is designed to provide visitors with an overview of my work and expertise.

Why SNDONG.CA?

I wanted to create a personal website that reflects my identity and showcases my work. The domain name "sndong.ca" is a combination of my name and the ".ca" domain, which represents Canada, where I am based. It serves as a professional online presence for me.

Learning

Through this project, I learned how to create a personal portfolio website using Django and Bootstrap. I also learned how to deploy a Django application on a web server. The website is designed to be responsive and user-friendly, allowing visitors to easily navigate through my projects and skills.

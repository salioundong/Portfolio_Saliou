Date: January 2023

Technologies used: GDScript

Concept

It was as part of an academic project which consisted of creating a complete video game.

Why Soccer?

I have always had a particular attraction to football. When I talk about football, I'm talking about modern football better known in Europe, but in Canada they call it soccer. In short, I decided to make my game and represent some concepts like ball control, goal detection, etc. You may have more details by visiting my Github with the source code link at the top.

Learning

Through this project, I learned a new programming language which is GDScript : an an object-oriented programming language designed to Godot. The project could be done with C#, but as I had already learned this language, I preferred the novelty and doing it with GDScript. which is also simpler than C# with Godot.

This is a small overview of the purpose of the project. The robot had a long journey to make.

Demo

Date: December 2022

Technologies used: C++, Arduino IDE, mBot Ranger


The robot itself


It's includes:

  • Motors with encoder
  • Aluminum chassis
  • Ultrasonic sensor
  • Line sensor
  • Temperature, light, power sensor
  • Buzzer
  • Motherboard compatible with the Arduino Mega

Particularities and learning

Although I have always had little interest in electronics in general, I still enjoyed my familiarization with the mBot Ranger of the company MakeBlock. Because it allowed me to play with the robot to react to sound intensity, light detection, obstacle avoidance, line following which consists of guiding the device along a trajectory defined by a line which is usually black in color on a pale background. Line following is also a fairly common task in robotics competitions as well as academic projects.

I learned and used all these mechanisms in small separate projects and then integrated them all into one big project shown in the demo at the top of the page. You can see that the robot has had a little of misery at a given moment to continue its trajectory because of a small problem with line following. Which also led to the intervention of my hand to put the robot back on the line.

I unfortunately lost the source code because I changed computer in the meantime and at the time I didn't use Github to push my projects forward.

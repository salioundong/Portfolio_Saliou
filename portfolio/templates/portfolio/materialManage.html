MaterialManage


Complete mobile application for equipment borrowing management.


Concept

It was as part an academic project and I had to carry out a concrete mobile application development project. The application is CRUDL type and I did:

  • Design (data model, user interfaces, architecture)
  • Programming (logic, user interfaces, database)
  • Documentation

I played with the authentification as can be seen with the image next to it. And for each entity that I had to do, I implemented the creation of the entity, the modification, the reading as well as its deletion from the list abd the database hence the CRUDL type.


Learning

I had to learn React Native which is an open source mobile application framework created by Facebook. For my native gateway, I used Expo and the mobile application Expo Go to test my code on my mobile device


Date: December 2023

Technologies used: JS, React, JSX, Expo Go

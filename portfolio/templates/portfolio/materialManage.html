Date: December 2023

Technologies used: JS, React, JSX, Expo Go

Source code

Complete mobile application for equipment borrowing management.


Concept

It was as part an academic project and I had to carry out a concrete mobile application development project. The application is CRUDL type and I did:

  • Design (data model, user interfaces, architecture)
  • Programming (logic, user interfaces, database)
  • Documentation

I played with the authentification as can be seen with the image next to it. And for each entity that I had to do, I implemented the creation of the entity, the modification, the reading as well as its deletion from the list and the database hence the CRUDL type.


Learning

J'ai eu à apprendre React Native qui est un framework d'applications mobiles open source créé par Facebook. Pour ma passerelle native, j'ai utilisé Expo et l'application mobile Expo Go pour tester mon code sur mon appareil.
